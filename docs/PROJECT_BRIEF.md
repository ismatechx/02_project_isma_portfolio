# Project Brief — Ismail Uthuman Portfolio

## Summary

A fully custom, live portfolio website built to showcase Ismail Uthuman's expertise as a Senior Business Intelligence Analyst. The site is hosted on GitHub Pages, connected to a real database, and features interactive embedded Tableau dashboards — demonstrating both BI skills and technical breadth.

---

## Problem Statement

Off-the-shelf portfolio platforms (LinkedIn, Tableau Public alone) don't allow full control over design, branding, or the ability to integrate multiple tools into one unified experience. A dedicated custom portfolio was needed that:

- Presents work history, skills, and certifications in a professional, memorable way
- Embeds live Tableau dashboards so hiring managers can interact with the actual work
- Captures inbound interest via a functional contact form backed by a real database
- Serves as a persistent, shareable link for job applications and networking

---

## Solution

A single-page portfolio website built from scratch with:

| What | How |
|---|---|
| Custom dark UI | Pure HTML/CSS with teal accent — no templates |
| Live hosting | GitHub Pages — auto-deploys on every update |
| Contact form with DB | Supabase (PostgreSQL) — all messages stored |
| Tableau dashboards | Embedded live via Tableau Public Embed API |
| Animations | CSS keyframes + IntersectionObserver scroll reveal |

---

## Key Features

- **Hero section** — animated background, floating KPI metric cards, typewriter role title
- **Experience timeline** — 4 roles across Zuci Systems and Radiare Software Solution
- **Skills section** — categorised by BI Tools, Programming & Databases, Cloud & ETL
- **5 Project showcase** — 3+2 grid layout, gradient banners, live Tableau embeds for public projects
- **Certifications & Awards** — 4 Microsoft/Tableau certs, 4 company awards
- **Contact form** — real database backend, form submissions stored in Supabase
